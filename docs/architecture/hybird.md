# 混合开发(Hybird)

- 原生与 Web 技术混合开发的模式

## 三种形式

### 1. 基于 Web 的解决方案

- Cordova、微信浏览器等
- Web 的性能受到浏览器内核的影响

### 2. 非基于 WebUI 但业务逻辑基于 Java 的解决方案

- React Native、Weex、Flutter

### 3. 基于 Web 且 UI 层与逻辑层分离的解决方案

- 各类小程序
- 运行效率比纯 Web 要高

## 动态化技术

- 内容数据和页面外观都不是固定的，都是随着服务器端的更新而更新，用户可以及时看到最新的内容

### 优点

- 可以快速迭代，动态更新
- 跨平台

### 缺点

- 性能
- 抹平平台差异有许多坑
