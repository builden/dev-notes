# git commit 规则

## 工作流 Git Flow

## 提交类型

### feat: 功能

- feat: add Message & NoticeBar

### fix: 问题修复

- fix: remove hasArrow

### refactor: 重构

- refactor: make checkbox controlled

### style: 更新样式

- style: format scss

### chore: 杂事、日常事务

- chore: upgrade react react-dom to v16.12.1

### pref: 性能

- pref: optimize styles of checkbox and radio

### test: 测试用例

- test: update test case

### docs: 更新文档

- docs: update api
